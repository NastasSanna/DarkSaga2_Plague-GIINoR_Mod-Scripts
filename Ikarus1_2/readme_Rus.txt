В зависимости от версии Готики (1 или 2) выберите G1 или G2 версии папок и файлов Ikarus_Const.d.

Порядок компиляции файлов пакета ([X] - версия Готики):
	1. Файл Ikarus_Const_G[X].d
	2. Папка EngineClasses_G[X]
	3. Ikarus.d (не зависит от версии)
Файл Ikarus_Doc.d компилировать НЕ НУЖНО.

Ikarus.d 				содержит все функции Ikarus.
Ikarus_Const_G[X].d		содержит константы, характерные для Готики версии [X].
Классы					позволяют находить описание свойств.
Ikarus_Doc.d			содержит подробную информацию об использовании этого пакета.

Файл Float.d содержит функции для работы с числами с плавающей запятой в формате Ikarus (zREAL).
Рекомендуется, но не представляют собой необходимое дополнение к Ikarus.
Пояснения по использованию находятся в самом файле.
Файл должен быть скомпилирован после Ikarus.d.
